1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It defines the structure of a webpage using elements, tags, and attributes. Every website you visit is built using HTML at its core.

Key Points:

HTML stands for HyperText Markup Language.

HyperText means text with links to other pages.

Markup Language refers to the system of tags used to define content.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

2. Structure of an HTML Document

Every HTML document has a basic structure:

<!DOCTYPE html> – Declares the document type.

<html> – Root element.

<head> – Contains metadata, links to stylesheets, and the title.

<body> – Contains all visible content on the webpage.

Mini Tip: Always include <meta charset="UTF-8"> in <head> to avoid encoding issues.

3. Common HTML Tags

Headings: <h1> to <h6>

Paragraphs: <p>

Links: <a href="url">Link</a>

Images: <img src="image.jpg" alt="description">

Lists: <ul>, <ol>, <li>

Tables: <table>, <tr>, <td>, <th>

Example – A mini profile card:

<h2>John Doe</h2>
<p>Web Developer</p>
<ul>
  <li>Email: john@example.com</li>
  <li>Phone: 123-456-7890</li>
</ul>
<img src="john.jpg" alt="John Doe">

4. Attributes

Attributes provide extra information about elements.

href for links

src for images

alt for accessibility

id & class for styling or scripting

Example:

<a href="https://example.com" target="_blank">Visit Example</a>

5. HTML Forms

Forms collect user data.

<form action="/submit" method="post">
  Name: <input type="text" name="username">
  Email: <input type="email" name="email">
  <button type="submit">Submit</button>
</form>


Short Notes:

action defines where data is sent.

method="post" hides user data, method="get" appends data to URL.

6. HTML Best Practices

Use semantic tags: <header>, <footer>, <article>, <section>

Keep code clean and indented.

Use alt attributes for images for accessibility.

Validate HTML using W3C validator.

7. Example Mini-Project

Create a Personal Portfolio Page

Sections: About, Skills, Projects, Contact

Use headings, paragraphs, images, and links.

Add lists for skills and projects.

8. Quick Tips

HTML alone cannot style pages; CSS handles styling.

HTML is not a programming language, it’s a markup language.

Learning HTML is the first step to becoming a web developer.
